# Projekt Lorenz w Julii

W tym repozytorium znajduje się notatnik Jupyter z kodem w języku Julia.

## Instrukcja uruchomienia

Aby uruchomić kod:
1. Upewnij się, że masz zainstalowane środowisko **Jupyter** (np. Jupyter Notebook, JupyterLab) oraz język **Julia**.
2. W środowisku Julia musi być zainstalowany pakiet `IJulia` (kernel umożliwiający uruchamianie kodu Julii w Jupyterze).
3. Otwórz terminal, przejdź do folderu z projektem i uruchom środowisko, np. wpisując:
   ```bash
   jupyter notebook lorenz_julia_numeryka.ipynb
   ```
4. Otwórz plik w oknie przeglądarki i po kolei uruchamiaj komórki.
